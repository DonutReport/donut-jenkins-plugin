Additional custom name/value pairs to be displayed in the report. Both the name and value are tolerant of space characters, i.e.


        App Version=1.0
        Comment=This is some comment with space characters

Both environment variables and maven properties can be specified for the value field, i.e.


        Path=$PATH //Env var
        App Version=$app.version //Maven property

Note: The plugin only resolves maven properties specified in the root pom.xml file.
